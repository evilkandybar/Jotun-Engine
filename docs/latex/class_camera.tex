\subsection{Detailed Description}
Defines a camera for use in rendering. 

The camera inherits from \hyperlink{class_input_handler}{Input\-Handler} so it can move in response to player input and inherits from \hyperlink{class_transformable}{Transformable} so it has a position, rotation, and the like

handles a key press for a camera 

If the key is W, the camera begins moving forward If the key is A, the camera begins moving left If the key is S, the camera begins moving back If the key is D, the camera begins moving right If the key is R, the camera begins moving up If the key is F, the camera begins moving down

All movements should be relative to the camera's alignment, but are actually relative to the world's alignment

This function is called from the key callback specified in main.\-cpp
