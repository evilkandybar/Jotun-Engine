\subsection{Detailed Description}
Defines a 3\-D mesh. 

This class can load and render and static mesh. Animation support coming soon

When you change mesh, call mesh.\-bind() When you actually want to draw the mesh, call mesh.\-draw() When you are done with the mesh, call mesh.\-disable()

Note that this isn't necessary until I have V\-B\-Os working

For now, just call \hyperlink{class_mesh_a996a8668fa2ca7d95d6d10744c833bc8}{draw()}
